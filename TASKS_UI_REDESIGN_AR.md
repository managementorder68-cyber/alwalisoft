## ✅ الميزات الجديدة

### 1. عناوين الأقسام
```tsx
<div className="flex items-center gap-2">
  <div className="w-1 h-6 rounded-full bg-gradient-purple" />
  <h2>المهام المتاحة</h2>
  <span className="badge">(5)</span>
</div>
```

### 2. فلترة ذكية
```tsx
// المهام المتاحة فقط
tasks.filter(task => !task.isCompleted)

// المهام المكتملة فقط
tasks.filter(task => task.isCompleted)
```

### 3. عرض شرطي للأقسام
```tsx
// إخفاء القسم إذا كان فارغاً
{tasks.filter(...).length > 0 && (
  <div>...</div>
)}
```

### 4. تأثيرات انتقالية
```css
transition-all duration-300
hover:shadow-lg hover:shadow-purple-500/20
```
